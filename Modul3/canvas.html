Canvas

Kesimpulan Hasil Ujicoba:
Canvas pada HTML5 digunakan untuk menggambar berbagai objek grafis secara dinamis menggunakan JavaScript. Berbeda dengan SVG yang berbasis vektor, canvas bekerja dengan sistem pixel-based sehingga cocok untuk membuat animasi, game, grafik interaktif, dan manipulasi gambar secara langsung di dalam browser. Dengan menggunakan file JavaScript terpisah seperti main.js, kode program menjadi lebih rapi dan mudah dikelola.
