    Kesimpulan Hasil Uji Coba:
    Video lokal menggunakan tag <video> dapat diatur langsung di dalam HTML. Kita bisa menambahkan atribut seperti controls untuk menampilkan tombol play/pause, autoplay agar video diputar otomatis, dan muted untuk mematikan suara. Video lokal berasal dari file yang disimpan di komputer atau server kita sendiri. Sedangkan video YouTube biasanya ditampilkan menggunakan tag <iframe>. Dengan cara ini, video diambil langsung dari server YouTube sehingga kita tidak perlu menyimpan file videonya di server. Jadi, penggunaan <iframe> lebih praktis untuk menampilkan video dari internet karena kita hanya perlu menempelkan link video tanpa harus mengunggah file video sendiri.
